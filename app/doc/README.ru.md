В игровом движке код — лишь малая часть работы. Остальное — это ассеты и пайплайн: материалы, блюпринты, ландшафт, небо, UI, скелетная анимация, упаковка, производительность. FreeUltraCode — это coding-агент в стиле Claude Code / Codex / Gemini, перестроенный вокруг этой реальности: он понимает концепции игровых движков, генерирует весь спектр игровых ассетов (изображения, 3D-модели, 2D-анимацию спрайтов, атласы, аудио, риггинг, видео) и направляет рутинную работу через бесплатные или недорогие каналы, чтобы премиальная квота шла туда, где она важна.

## Зачем нужен FreeUltraCode

Сегодня ИИ достаточно хорош, чтобы писать большую часть кода самостоятельно. Роль программиста смещается к описанию задач, проверке результата и оркестрации агентов. Но игра — это не только код. Игровой движок полон материалов, блюпринтов, ландшафта, неба, UI, скелетной анимации, упаковки и тюнинга производительности — и большинство универсальных coding-агентов в этом ничего не понимают.

FreeUltraCode берёт агента в стиле Claude Code / Codex / Gemini и глубоко адаптирует его под разработку игр:

- **Говорит на языке игровых движков.** Агент подготовлен концепциями игровой разработки и может рассуждать о материалах, блюпринтах, ландшафте, освещении, UI (UMG и других), скелетной анимации, сборке/упаковке и оптимизации производительности.
- **Генерирует все типы ассетов, нужные игре.** Изображения, 3D-модели, 2D-анимация спрайтов, атласы спрайтов, аудио, скелетный риггинг и видео создаются в одном интерфейсе и управляются единым рабочим процессом агента.
- **Встроенная команда экспертов по разработке игр.** Более 40 специализированных ролей (технический директор, программист геймплея/ИИ/сети/инструментов, дизайнер уровней/экономики, арт- и аудиодиректоры, QA, релиз-менеджер и др.) для Unity, Unreal, Godot и Web.
- **Бережёт премиальную квоту для главного.** Направляйте рутину через бесплатные, пробные или недорогие каналы, а ключи, настройки и историю храните локально.

## Возможности

### Чат для разработки игр

- Запрашивайте код геймплея, интеграцию движка, логику шейдеров/материалов, скрипты сборки, расследование багов, рефакторинг, тесты и release notes.
- Работайте над проектами Unity, Unreal, Godot или Web — агент рассуждает о концепциях движка, а не только о файлах.
- Добавляйте пути к файлам или перетаскивайте файлы в область ввода.
- Смотрите потоковые ответы, логи, ссылки на файлы и резюме в одном чате.
- Продолжайте работу уточняющими запросами в той же сессии.

### Генерация игровых ассетов

Любой тип ассета, нужный игре, можно сгенерировать в одном интерфейсе, применить к проекту и передать обратно модели программирования — всё в одной истории. Каждый генератор работает через настроенный вами провайдер.

| Ассет | Что создаёт | Режим |
| --- | --- | --- |
| Изображения | Концепт-арт, макеты UI, иконки, постеры, текстуры, референсы | `/image`, `/img`, `/draw`, `/生图` или `/image-mode-start` |
| Графы ComfyUI | Редактируемые узловые пайплайны изображений | `/comfyui-mode-start` |
| 2D-спрайты | Игровые спрайты, кадры последовательностей, spritesheets | `/sprite` или `/sprite-mode-start` |
| 3D-модели | Пропсы, персонажи, меши сцен, блокауты | `/mesh-mode-start` (поиск через `/mesh-search`) |
| Музыка | BGM, саундтрек, музыкальные клипы | `/music` или `/music-mode-start` |
| Речь | Реплики и закадровый текст | `/speech-mode-start` |
| Видео | Видеоклипы и анимированные ассеты | `/video` или `/video-mode-start` |

Агент сначала улучшает ваш prompt, отправляет его настроенному провайдеру и показывает результат в потоке чата. Выходите из любого режима соответствующей командой `*-mode-end`.

### Команда экспертов по разработке игр

FreeUltraCode включает более 40 специалистов по разработке игр, которых агент привлекает автоматически в зависимости от задачи:

- **Специалисты по движкам** для Unity, Unreal, Godot (GDScript / C# / GDExtension / шейдеры) и Web.
- **Программирование**: технический директор, lead-/движковый/геймплейный/ИИ-/сетевой/инструментальный/UI-программисты.
- **Дизайн**: дизайнеры геймплея, уровней, экономики, live-ops и нарратива.
- **Арт и аудио**: директора и специалисты, VFX, sound design, аудиорежиссура.
- **Продакшн, качество и релиз**: продюсер, QA lead/тестировщик, devops, безопасность, локализация, релиз-менеджер.

Настройте активный движок, режим council и включённых экспертов в **Settings**.

### Динамический рабочий процесс (/ultracode)

Для сложных многоэтапных задач программирования `/ultracode <задача>` мгновенно генерирует специализированный исполняемый harness и сразу запускает его. Визуальный холст не требуется.

- Опишите задачу на естественном языке — планировщик строит harness с параллельными подагентами, состязательной проверкой и воротами приёмки.
- Шесть внутренних стратегий выбираются автоматически: классифицировать-и-действовать, веер-и-синтез, состязательная-проверка, генерировать-и-фильтровать, турнир, цикл-до-завершения.
- Каждый запуск полностью протоколируется в `.fuc-run/<run-id>/` с реестром задач, событиями, вердиктом и итоговым результатом.
- Запуск из десктоп-приложения или через CLI: `fuc ultracode "<задача>" --json --interactive --cwd <workspace>`.
- Нулевая конфигурация — используются локальные учётные данные `claude` CLI.

#### Free Auto — Автоматическое переключение каналов

Канал **Auto** (`freecc:auto` в меню Channel) автоматически направляет каждый запрос через лучший доступный бесплатный канал без ручного переключения.

- Перебирает все настроенные бесплатные каналы, автоматически пропуская те, что достигли ограничений (429) или возвращают ошибки upstream (5xx).
- Отслеживает время охлаждения каждого канала с откатом: после ошибки канал приостанавливается перед повторной попыткой.
- Поддерживает опциональную перезапись модели, чтобы все авто-маршрутизируемые запросы использовали одну модель.
- Если все каналы исчерпаны, возвращает 503 с журналом ошибок для диагностики.

#### Цепочка поставщиков: DeepSeek → CodeX

С `/ultracode` harness может автоматически связывать нескольких поставщиков между шагами плана. Типичный сценарий: DeepSeek создаёт быстрые черновики с низкими затратами, CodeX дорабатывает результат до финального качества.

- **Динамический план harness** поддерживает перезапись `model` для каждого шага — назначьте DeepSeek для мозгового штурма/классификации, а CodeX/Gemini для реализации/проверки.
- **Совместимость с cc-switch**: FreeUltraCode читает конфигурацию `cc-switch` CLI; любой поставщик, уже настроенный для Claude Code, сразу доступен для шагов ultracode.
- Стратегия **веер-и-синтез** распараллеливает DeepSeek-воркеров по независимым подзадачам, затем консенсус-гейт (CodeX) синтезирует и проверяет результаты.

#### Выбор канала с учётом скорости

Auto-канал бесплатного прокси приоритизирует каналы на основе сигналов реальной доступности:

- **Учёт ограничений скорости**: каналы с 429 охлаждаются на 30+ секунд перед повтором, избегая бесполезных попыток.
- **Быстрый отказ при ошибках**: неповторяемые ошибки (4xx аутентификация, 5xx падение upstream) отслеживаются с охлаждением; Auto-роутер их пропускает.
- **Бюджет времени соединения**: каждая попытка ограничена таймаутом upstream; Auto-роутер не блокируется на одном медленном upstream.
- **Естественный порядок по отзывчивости**: успешные каналы пробуются первыми; ошибочные каналы отодвигаются в конец списка.

Эти возможности обеспечивают устойчивость запусков `/ultracode`, даже когда отдельные бесплатные поставщики медленны, ограничены или временно недоступны.

### Генерация игровых ассетов

Режимы ассетов превращают поле чата в поверхность генерации ассетов, сохраняя ту же историю. Полезны для макетов UI, иконок, текстур, спрайтов, 3D-моделей, аудио и видео перед возвратом к коду. В примере ниже используется режим изображения; режимы спрайтов, мешей, музыки, речи и видео работают так же со своими командами `*-mode-start`.

1. Откройте **Settings** -> **Images** (или соответствующий раздел ассета), выберите провайдера по умолчанию и заполните API key, Account ID, Base URL или локальный endpoint ComfyUI.
2. В сессии чата введите `/image-mode-start`. Можно запустить режим и сразу сгенерировать в одном сообщении:

```text
/image-mode-start стилизованная текстура каменной стены для фэнтези-подземелья, бесшовная, 1024x1024
```

3. Пока режим включен, обычные сообщения генерируют ассеты вместо правок кода. Селектор **Channel** переключается на провайдеров ассетов.
4. Опишите нужный ассет. FreeUltraCode сначала улучшает prompt через модель программирования, затем отправляет его настроенному провайдеру.

<p align="center">
  <img src="images/生图/session-2026-06-07-2351.png" alt="Режим ассетов генерирует элементы в той же сессии FreeUltraCode" width="720">
</p>

5. Отправьте `/image-mode-end`, чтобы вернуться к каналу и модели программирования. Для одного ассета без постоянного режима используйте `/image`, `/img`, `/draw`, `/生图`, `/sprite`, `/music` или `/video` с prompt.

## Как это работает

```text
Запрос пользователя
    |
    v
Chat composer
    |
    +--> выбранные runtime / канал / разрешения / workspace
             |
             +--> API провайдера, локальный CLI или локальный free-channel proxy
                        |
                        +--> потоковый вывод, лог инструментов и история чата
```

